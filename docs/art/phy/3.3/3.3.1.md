## 什么是电流

电流是由电荷定向移动形成的流。物体中有电流，即物体中的电子或者离子在流动🏊；电流的大小就是电荷的流动速率，即电荷通过物体的快慢。电子在介质中运动的速率就是光在介质中的速率，因此要改变速率，就只能改变电荷通过的数量，即 $I = \frac{Q}{t}$ 。

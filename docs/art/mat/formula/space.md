# 立体几何

## 体积

$V_{棱柱}=Sh$

$V_{棱锥}=\frac13Sh$

$V_{棱台}=\frac13h(S'+\sqrt{S'S}+S)$

$V_{圆柱}=\pi{r^2}h$

$V_{圆锥}=\frac13\pi{r^2}h$

$V_{圆台}=\frac13\pi{h}({r'}^2+r'r+r^2)$

$V_{球}=\frac43\pi{R^3}$

# 不等式

## 重要不等式

$a^2+b^2≥2ab$

因为 $a, b\in \mathbf{R}, (a-b)^2≥0$，当且仅当 $a=b$ 时，等号成立，所以 $a^2+b^2-2ab≥0$

## 基本不等式

$\sqrt{ab}≤\frac{a+b}2$
